# 向量

1. 向量vector类，其中封装了方法magnitude返回一个向量的长度。

2. normalized方法可以返回一个单位向量。

3. 三个标准向量

   * vector.right，X轴单位向量
   * vector.up，Y轴单位向量
   * vector.forward，Z轴单位向量

4. 向量加法（减法）：可以用减法计算两个对象之间的距离

5. 向量乘法

   * 算术乘法：向量**a**与2相乘。

   * 点乘：全部相乘

6. 向量夹角：vector.signedAngle(b,a,vector3.forward);向量a逆时针旋转多少个角度到达b所指向的方向。当调用的方法为angle时表示为无符号数。
